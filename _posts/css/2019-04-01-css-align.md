## vertical-align : 테이블
```html
<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8" />
<title></title>
<style>
	table {
    	width : 100%;
        height : 100px;
        border-collapse: collapse;
    }
    table td {
    	border : 1px solid red;
    }
	table td:nth-child(1){
    	vertical-align : top ;
    }
    table td:nth-child(2){
    	vertical-align : middle ;
    }
    table td:nth-child(3){
    	vertical-align : bottom ;
    }
</style>
</head>
<body>
	<table>
    	<tr>
        	<td>a</td>
            <td>a</td>
            <td>a</td>
        </tr>
    </table>
</body>
</html>
```
