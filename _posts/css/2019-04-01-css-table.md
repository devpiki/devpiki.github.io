```html

<html>
<head>
	<title></title>
	<style>
		table.test1 {
            border-collapse: collapse;
            border : 1px solid red;
		}
        /*table.test1 tr {
        	border : 1px solid red;
        }
        table.test1 th, table.test1 td {
        	border : 1px solid red;
        }*/
```

## text-align, vertical-align
```html
<!DOCTYPE html>
<html>
<head>
	<title></title>
	<style>
		table.test1 {
            border-collapse: collapse;
            width : 100%;
            height : 500px;
		}
        table.test1 th, table.test1 td {
        	border : 1px solid red;
        }
        table.test1 td:last-child {
        	text-align : right;
            vertical-align : bottom;
        }
```
