## em 과 rem의 차이
```html
<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8" />
<title>단위 테스트</title>
<style>
	html {font-size:10px;}
	#emtest {font-size:2em;}
	#emtest p {font-size:1em;}
	#remtest {font-size:2em;}
	#remtest p {font-size:1rem;}
</style>
</head>
<body>
  <div id="emtest">
  	<!-- 결과 : 20px -->
    <p> text123 </p>
  </div>
  <div id="remtest">
  	<!-- 결과 : 10px -->
    <p> text123 </p>
  </div>
</body>
</html>
```
