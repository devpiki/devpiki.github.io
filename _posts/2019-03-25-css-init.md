## External Style Sheet
```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>test</title>
  <!-- 외부 스타일 시트 -->
  <link rel=”stylesheet” type=”text/css” href=”aaa.css”>
</head>
<body>
외부 스타일 시트 (External Style Sheet)
</body>
</html>
```

## Internal Style Sheet
```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>test</title>
  <!-- 내부 스타일 시트 -->
  <style type=”text/css”>
  <!--
  body {
      display:block;}
  //->
  </style>
</head>
<body>
내부 스타일 시트(Internal Style Sheet)
</body>
</html>
```

## Inline Styles
```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>test</title>
  <style type=”text/css”>
</head>
<!-- HTML 태그 내에 스타일 지정 -->
<body style=”color:red;”>
HTML 태그 내에 스타일 지정(Inline Styles)
</body>
</html>
```
