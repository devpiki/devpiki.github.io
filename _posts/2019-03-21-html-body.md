### 1-1. 기본 문법
```html
<!DOCTYPE html>
<html lang="ko">
<!-- bodytest.html --> 
<head>
    <title>body 태그 연습</title>
    <meta charset="UTF-8">
</head>
<body>
body 태그 영역
</body>
</html>
```

### 1-2. body 속성값
```html
<!DOCTYPE html>
<html lang="ko">
<!-- bodytest.html --> 
<head>
    <title>body 태그 연습</title>
    <meta charset="UTF-8">
</head>
<body 
    background="/assets/background.jpg"
    bgcolor="red"
    text="blue"
    topmargin=10
    leftmargin=20
>
body 태그 영역
</body>
</html>
```
